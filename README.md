# Training Courses Registration System

A PHP and MySQL web application for managing training courses and student registrations.  
The system provides an Arabic RTL interface for visitors and an admin dashboard for managing courses and registrations.

---

## 🌐 Project Overview

This project is designed for an educational academy that offers training courses.  
Users can browse available courses, register for a course, and access a simple Arabic interface.  
The admin can log in to manage courses and view submitted student registrations.

---

## 🚀 Features

- Arabic RTL user interface
- Responsive homepage
- Training courses listing
- Course registration form
- Admin login page
- Admin dashboard
- Course management
- Registration management
- Clean and simple design
- Built for local XAMPP environment
